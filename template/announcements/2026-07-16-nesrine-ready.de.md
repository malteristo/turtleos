# Was neu ist

Dein Kanal läuft auf einer klareren, stabileren Version. Dein Raum ist weiterhin **nur deiner** — nichts wird woanders geteilt.

## Wer ist wer?

**River** ist kein zweiter Bot — so heißt die **Oberfläche hier im Kanal**: kurze Bestätigungen und Buttons unten. Fürs richtige Gespräch brauchst du einen **Wirbel** (Thread).

**Turtle** ist dein Gesprächspartner **im Wirbel** — warm, praktisch, auf Deutsch wenn du Deutsch schreibst.

## So geht's

1. Unten auf **`new eddy`** klicken → Wirbel öffnet sich.
2. Eine Nachricht schreiben → Turtle antwortet dort.
3. Wenn du magst: **checkpoint** im Wirbel — Turtle schreibt eine kurze Notiz zu dem Gespräch. Über mehrere Gespräche hinweg kann Turtle Themen vorschlagen und fragen, ob sie merkwürdig bleiben sollen.
4. Optional: **Fresh Eyes** in der Flow-Bibliothek — ein ruhiger Blick auf das, was gerade in Bewegung ist. Kein Druck, nichts abarbeiten.

**Optional:** Im Wirbel auch **Feedback**, wenn etwas gut war oder nicht geklappt hat. Du entscheidest, ob und wann.

Keine Hausaufgaben. Probier es, wenn du Lust hast.
